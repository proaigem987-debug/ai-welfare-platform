About Project

This project aims to develop an intelligent platform that assists citizens in identifying appropriate welfare schemes based on their socio-economic profile using AI techniques.

Problem Statement

Many welfare schemes remain underutilized due to lack of awareness, complex eligibility rules, and absence of a unified digital platform. This leads to ineffective benefit distribution.

Key Features

  • User Registration & Login
  • AI-based Eligibility Check
  • Scheme Recommendation
  • Admin Dashboard
  • Data Analytics

Technologies Used

  • HTML, CSS, JavaScript
  • Python & Machine Learning
  • Flask / Django (Backend)
  • GitHub Pages (Hosting)
